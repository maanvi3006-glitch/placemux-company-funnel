# PlaceMux Company Funnel

Project scaffold for the PlaceMux company-side funnel analysis (Phase 2 Task 3).

Structure and placeholder files created for data ingestion, preprocessing, analytics, visualization, and dashboard exports.

## How to use
- Install dependencies: `pip install -r requirements.txt`
- Populate `data/raw/` with source CSVs and run preprocessing scripts in `src/preprocessing`.
